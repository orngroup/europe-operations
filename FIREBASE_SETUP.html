CapEye — Firebase Login Setup

One-time setup · Takes about 20 minutes · No technical knowledge needed
1 Go to Firebase Console
  • Open console.firebase.google.com in your browser
  • Click on your project: capeye-autocapital
2 Enable Email/Password Sign-In
  • In the left sidebar, click Build → Authentication
  • Click the Sign-in method tab
  • Click Email/Password
  • Toggle Enable to ON
  • Click Save

4 Upload the Files to GitHub
  • Go to your GitHub repo: github.com/orngroup/capeyecommandcentre
  • Click Add file → Upload files
  • Drag in all files from the CapEye_WithLogin.zip you received
  • Click Commit changes
  • Wait 2 minutes for GitHub Pages to update
5 Test It
  • Open your GitHub Pages URL and go to login.html
  • Sign in as Keith: email keith@autocapital.co.uk / password keithac
  • You should land on the dashboard with Keith's name in the top right
  • Try another login — e.g. Germaine (Sales) — and verify the nav shows correctly
  • The name in the top-right corner is clickable to sign out
